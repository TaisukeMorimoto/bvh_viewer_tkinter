# BVH Viewer by tkinter

# requirements
- python 3.9
- pipenv

# Steps
1. create data folder in root.
2. put bvh file in data folder.
3. `pipenv install`
4. `pipenv shell`
5. execute python command.
    ```commandline
        python src/usecase/bvh_viewer_usecase.py data/xxx.BVH
    ```
6. will display graphical view.
